## 🌿 TRABALHANDO COM BRANCHES

### Criando e Mudando entre Branches
```bash
# Ver branches existentes
git branch

# Criar nova branch
git branch nova-feature

# Mudar para branch
git checkout nova-feature

# Criar e mudar (atalho)
git checkout -b nova-feature

# Voltar para main
git checkout main
